Finds the best combined driving and walking path between two nodes.

Parameters
gPointer to the graph.
startStarting node.
endEnding node.
max_walkingThe maximum walking time allowed.
messageReference to a message string describing the status.
alternativeBoolean flag to indicate if an alternative route is needed.
avoid_nodesList of nodes to avoid.
avoid_edgesList of edges to avoid.
Returns
Pair of paths representing the driving and walking routes.
Note
Time Complexity:
  • Best-case: O((V + E) log V) if a parking spot is found quickly.
  • Average-case: O((V + E) log V + P V), where P is the number of parking nodes (it is significantly smaller than V).
  • Worst-case: O((V + E) log V + V^2) if every node is a parking node.
